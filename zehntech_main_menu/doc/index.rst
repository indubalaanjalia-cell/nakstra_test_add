Main Menu Module
================================================================

The Main Menu Module provides a centralized and user-friendly interface for accessing all major Odoo applications. Designed to enhance navigation and productivity, it offers customizable widgets such as live date/time displays and administrator-managed announcements. Users can bookmark frequently used menus and external links for quicker access, while search and sort options make it easy to manage bookmarks efficiently. This module improves the overall user experience by consolidating essential tools and information into a single, streamlined dashboard.

**Key Features**
================================================================

- **Centralized Main Menu for Quick Access**: Provides a unified dashboard for accessing core Odoo modules quickly and efficiently. Enhances user productivity by reducing navigation time across different applications. 
- **Add and Manage Bookmarks for Frequently Used Menus**: Allows users to bookmark frequently accessed internal menus and external links for quicker access.Simplifies navigation by letting users organize their favorite destinations in one place.
- **Bookmark Management with Search and Sorting**: Easily search and sort bookmarks to quickly find frequently used links and menus.Improves user experience with organized, customizable access to bookmarked items.
- **Show Current Date and Announcements on Top**: Displays the current date and important admin announcements prominently at the top of the dashboard.Keeps users informed and up to date as soon as they access the main menu.

**Installation**
================================================================

1. Download the module from the Odoo App Store or clone the repository.
2. Place the module in your Odoo addons directory.
3. Update your Odoo instance to include the new module.
4. Install the module through the Odoo interface.

**How to use this module:**
================================================================

1. Navigate to the Main Menu app in your Odoo dashboard.
2. **Access the Main Menu App**: Navigate to the Main Menu from your Odoo dashboard or the top-level menu. It opens a centralized screen that gives quick access to all core modules and tools.
3. **Centralized Main Menu Dashboard** : Upon opening, youâ€™ll see a visually organized dashboard with tiles for commonly used Odoo apps (like Sales, Inventory, CRM, etc.), along with customizable widgets like Current Date/Time and Admin Announcements.
4. **Configure Widget Visibility (Optional)**: Go to Settings > General Settings, scroll to the Main Menu section, and enable or disable features like date display or announcements visibility for users.
5. **Use the Bookmarks Section**: On the Main Menu screen, locate the Bookmarks section. It allows users to store frequently accessed internal menus or external URLs for quick access.
6. **Add a Bookmark**: Click on "Add Bookmark". Enter the Name, URL, and choose whether to open it in the current tab or a new tab. The new bookmark will appear in your personal list immediately.
7. **Manage Bookmarks**: Use the built-in search bar and sorting options to quickly find or organize your saved bookmarks. You can delete or edit bookmarks as needed.
8. **Admin Announcements Feature**: Admins can go to Settings > Announcements and create messages that are shown on the Main Menu dashboard. These are useful for company-wide notices or updates.
9. **Responsive & User-Specific UI**: Each user sees a personalized dashboard with only the bookmarks they've saved. The layout is mobile-friendly and designed for optimal performance.
